Commands

All available commands for Paperplane music bot

🎵 Music

play <query|url>
Play a song from YouTube, Spotify, or supported source
skip
Skip the currently playing track
stop
Stop playback and clear the queue
pause
Pause the current track
resume
Resume paused playback
queue
Display the current song queue
volume <1-100>
Set or view the playback volume
autoplay
Toggle automatic song recommendations when queue ends
nowplaying np
Show information about the current track

⚙️ System

prefix [new_prefix]
View or change the bot's command prefix
ping
Check the bot's latency
help
Display the help menu
uptime
Show how long the bot has been online
stats
Display bot statistics and information
